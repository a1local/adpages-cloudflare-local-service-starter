# Privacy

This starter is static and dependency-free.

It does not:

- collect analytics
- set cookies
- use local storage
- make client-side network calls
- load remote images, fonts, scripts, pixels, embeds, or stylesheets
- insert hidden backlinks
- store form submissions

The example quote CTA uses `mailto:` and the phone CTA uses `tel:`. UTM examples are visible URL parameters only; they are not captured, stored, or transmitted by JavaScript.

Cloudflare Pages may still provide platform-level request logs, security events, and deployment logs depending on the account configuration. Document those settings in the live site's privacy policy before publishing for a real business.

If you add analytics, forms, chat widgets, maps, call tracking, heatmaps, or remarketing tags, update this file before publishing.
